jquery.instagram.js by robedge

A simple plugin for integrating an instagram feed

This is a jQuery plugin created to wrap all of the API methods in the Instagram API. It is a work in progress and actively being developed.

Dependencies

jQuery 1.5+

Install

Include jQuery, Include the plugin, write code
